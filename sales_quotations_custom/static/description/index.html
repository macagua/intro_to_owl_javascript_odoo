This app is a tutorial in the form of code. The app will learn you how to create new widgets with OWL in Odoo.

Learn how to use OWL and create widgets

This app contains all the code you need to add a new OWL widget in Odoo.
All the code is documented inline with notes and extra information to understand how widgets work.

Contains

  • New widget "o_partner_order_summary" with all the code to build your own widget(s)
